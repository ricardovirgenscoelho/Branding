# RVC TECH - Branding Repository

This repository hosts public branding and identity assets for the RVC TECH domain.

## Purpose

This repository is used to publish publicly accessible branding files required for email identity validation, visual trust indicators, and domain reputation services.

## Files

* `logo.svg` → Official BIMI logo
* `index.html` → Public branding landing page
* `.nojekyll` → Disables Jekyll processing for GitHub Pages compatibility
* `CNAME` → Custom domain mapping for GitHub Pages
